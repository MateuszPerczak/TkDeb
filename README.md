# PyDeb
Debugger for tkinter application

## Python version:
3.8.2

## Used Libraries:

+ tkinter
+ traceback
+ typing

## Using debugger in tkinter app

```py
from PyDeb import Debugger
from tkinter import Tk

root = Tk()
Debugger(root)
root.mainloop()
```
## Keyboard shortcuts:

+ F2 - extended / normal mode
+ Esc - exit application
